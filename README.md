opendatacore
============

ODC is a powerful open source data processing engine designed to enable data collection, augmentation,
search, filtering and analytics.


Change Log
----------

V0.0.1 (Alpha 1)
Initial working build that supports content from feeds (polling source) and twitter (post adapter source).

V0.0.2 (Alpha 2)
Improved the solr schema and implemented facebook (polling source), added search basic search api.
Changed the relationships of Source to source-many-to-many-dataset.
